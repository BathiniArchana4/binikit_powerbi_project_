# Blinkit Sales Data Analysis Dashboard (Power BI) #

## 🛒 Project Overview ##
This Power BI dashboard provides an in-depth analysis of Blinkit's sales performance, customer satisfaction, and inventory distribution. The project transforms raw e-commerce grocery data into actionable insights, helping management make data-driven decisions to optimize operations and enhance customer experience. 

## 📊 Key Insights & Findings ##
Total Sales: The dashboard analyzes total revenue, identifying top-performing categories (e.g., Snacks, Beverages).
Outlet Performance: Medium-sized outlets and urban locations typically generate the highest total sales and customer engagement.
Customer Preferences: Low-fat products demonstrate significant sales, reflecting strong demand for healthy items.
Outlet Growth: Insights show steady sales growth over time, with tier 2 cities contributing significant volume.
Average Ratings: Overall customer satisfaction is analyzed, providing an average rating for products and outlets.

## ⚙️ Power BI Skills Demonstrated ##
Data Cleaning: Utilizing Power Query to fix data types, handle missing values, and structure the data.
DAX Measures: Creating dynamic measures (e.g., Total Sales = SUM(Item_MRP) or Average Rating = AVERAGE(Rating)).
Data Modeling: Linking tables (e.g., Sales, Outlet, Product) to enable interactive filtering.
Dashboard Designing: Creating a user-friendly, clean UI with filters (Slicers) for Year, Outlet Type, and Location. 

## 💡 Business Impact ##
Strategic Growth: Helps identify top-performing outlets and regions for expansion.
Operational Efficiency: Optimizes product inventory by identifying high-demand items.
Marketing Optimization: Enables targeted promotions based on location-wise customer behavior. 

## 🚀 How to Use the Dashboard ##
Slicers: Use the top panel to filter data by Outlet Location, Outlet Size, or Item Fat Content.
Interact: Click on any bar chart or pie chart segment to cross-filter the entire dashboard.
Insights: Analyze the KPIs to immediately see the overall performance at a glance.
